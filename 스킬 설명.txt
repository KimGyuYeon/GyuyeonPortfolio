html
표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹사이트 퍼블리싱을 하고 있습니다. 웹 표준을 위해 HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 이용해 전체 레이아웃을 설계합니다. 또한 웹 접근성 준수를 위해 키보드에 대한 대응과 적절한 대체 텍스트를 제공하고자 노력합니다.

css
크로스브라우징을 위해 안정적인 CSS 속성을 사용해서 HTML Element를 배치하고 디자인합니다. 레이아웃 제작에서 기존 방식인 float과 최신 트랜드인 flex 기법을 적재적소에 활용할 수 있습니다.

js
이미지 슬라이드, 마우스 휠에 반응하는 슬라이드 배너, 내비게이션, 탭 메뉴 등 웹 퍼블리싱에 필요한 제이쿼리는 플러그인을 사용하지 않고 순수 제이쿼리 코딩으로 제작할 수 있습니다. 또한 필수적인 자바스크립트 활용은 가능하며 주로 제이쿼리와 병행해서 활용합니다.
기본적인 문법 사용과 플러그인 사용이 가능하며 제이쿼리를 이용한 동적표현이 가능합니다.

반응형
반응형 웹 제작에 있어서 크로스브라우징을 위해 안정적인 기존방식인 float, position, Media Query를 활용한 반응형 웹 제작 경험을 가지고 있습니다. 모바일 웹 앱 제작의 경우에는 flex를 활용합니다.

git

공동 개발에서 효율적인 버전관리를 위해 git 명령어를 학습하여 브랜치 사용이 가능합니다

figma

피그마를 이용한 공동 작업이 가능하고 실시간으로 수정되는 부분들을 볼 수 있어서 즉각적인 대응이 가능합니다. 컴포넌트, 플러그인, 오토레이아웃 기능과 다양한 플러그인을 사용할 수 있습니다.

photoshop
웹 퍼블리셔 직군 특성에 맞게 Adobe Photoshop 기본 툴을 다루는 것이 가능합니다.

illustrator
웹 퍼블리셔 직군 특성에 맞게 Adobe Illustrator 기본 툴을 다루는 것이 가능합니다.

after effect
Adobe After Effects를 사용하여 간단한 자막 삽입, 도형툴, 효과넣기 등의 편집 등이 가능합니다.

pro

Adobe Premiere Pro를 사용하여 간단한 자막 삽입, 도형툴, 효과넣기 등의 편집 등이 가능합니다.
